# xingkailake1
兴凯湖国家地质公园地质遗迹数据管理系统旨在实现水体检测、土壤采样等环境数据与地图相结合，通过网页的方式在地图上进行展示，表达采样点的分布状况和数据的表格化展示。
前端采用的vue框架，使用的Javascript语言进行编写。本人负责前端的全部工作。
